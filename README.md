# 📊 DataDash - Admin Analytics Dashboard

Un panel de control analítico corporativo interactivo y moderno, diseñado para simular la visualización de métricas de ventas y comportamiento de usuarios en tiempo real. Este dashboard cuenta con múltiples visualizaciones gráficas integradas para analizar de forma rápida el rendimiento empresarial.

## 🚀 Características Principales

- **Tarjetas de KPI Dinámicas**: Monitoreo de ingresos, nuevos clientes y tasas de conversión con indicadores de tendencia (↑/↓).
- **Gráficos Interactivos Avanzados**:
  - *Evolución de Ingresos*: Gráfico de líneas mensual con gradientes suaves.
  - *Ventas por Categoría*: Gráfico semicircular (Doughnut).
  - *Métricas de Adquisición*: Gráfico de barras interactivo.
- **Simulación de Base de Datos**: Generación automática y fluida de datos para visualización de tablas transaccionales de clientes.
- **Filtros Temporales**: Ajuste de rango de fechas (7 días, 30 días, trimestre, año) que recalcula y re-escala instantáneamente todas las métricas simuladas.
- **Exportación Virtual**: Simulación de descarga de reportes en formato CSV con feedback de carga asíncrono.
- **Diseño de Interfaz "SaaS Professional"**: Tipografía corporativa (Inter), esquema de colores adaptado para analítica (Dark modern theme) e iconografía intuitiva.

## 🛠️ Tecnologías Empleadas

- **HTML5**: Estructura semántica avanzada y layout asimétrico.
- **CSS3**: Estilos orientados a interfaces tipo *SaaS*, con variables CSS, transiciones de estados y técnicas de diseño moderno (Flexbox y Grid Layout).
- **JavaScript (ES6+)**: Lógica robusta de simulación de negocio, cálculos dinámicos proporcionales de KPIs y manipulación reactiva del DOM.
- **Chart.js**: Biblioteca ligera e íntegramente implementada para renderizar todos los gráficos de manera interactiva.
- **Google Fonts**: Implementación de la tipografía *Inter* garantizando una alta legibilidad.

## ⚙️ Cómo Ejecutar el Proyecto Localmente

El proyecto consiste puramente en tecnologías Frontend estáticas y no necesita dependencias adicionales.

1. Clona este repositorio o descarga los archivos.
   ```bash
   git clone <url-del-repositorio>
   ```
2. Navega al directorio raíz del proyecto:
   ```bash
   cd api-dashboard
   ```
3. Para una experiencia óptima, se recomienda abrir el proyecto utilizando un servidor local. Si utilizas Visual Studio Code, herramientas como la extensión **Live Server** son ideales.
4. Si no dispones de un servidor local, simplemente abre el archivo `index.html` en tu navegador web de preferencia.
